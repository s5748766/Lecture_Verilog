# 예제 프로젝트

이 디렉토리에는 실습할 수 있는 다양한 예제 프로젝트가 포함되어 있습니다.

## 📁 예제 목록

### 1. hello_make
가장 기본적인 Make 예제
- 단일 C 파일 컴파일
- 기본 Makefile 구조 학습

### 2. simple_project
간단한 멀티 파일 프로젝트
- 여러 소스 파일 관리
- 헤더 파일 의존성
- 변수와 패턴 규칙 사용

### 3. library_example
정적/공유 라이브러리 생성
- 정적 라이브러리(.a) 생성
- 공유 라이브러리(.so) 생성
- 라이브러리 링크 방법

### 4. modular_project
모듈화된 대형 프로젝트
- 재귀적 Make
- 서브디렉토리 관리
- 설정 파일 분리

### 5. cross_platform
크로스 플랫폼 빌드
- Linux/macOS/Windows 지원
- 플랫폼별 설정
- 조건부 컴파일

## 🚀 사용 방법

각 예제 디렉토리로 이동하여 다음 명령을 실행하세요:

```bash
# 예제 디렉토리로 이동
cd examples/hello_make

# README 읽기
cat README.md

# 빌드
make

# 실행
./program

# 정리
make clean
```

## 💡 학습 순서

초보자는 다음 순서로 학습하는 것을 권장합니다:

1. `hello_make` - Make의 기본 개념
2. `simple_project` - 실용적인 프로젝트 구조
3. `library_example` - 라이브러리 생성과 사용
4. `modular_project` - 대형 프로젝트 관리
5. `cross_platform` - 이식성 있는 빌드 시스템

각 예제의 상세한 설명은 해당 디렉토리의 README.md를 참조하세요!
